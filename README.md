# Toby Math Academy

A student-led mathematics learning platform built by a Grade 10 student passionate about:

- BC School Mathematics
- AMC Competition Preparation
- Waterloo Math Contests
- Video-based Math Learning
- Peer Tutoring
- English-Chinese Math Vocabulary

## Planned Features

- BC curriculum math lessons
- AMC problem walkthroughs
- Competition roadmap
- Student accounts and learning dashboard
- Video lessons
- Bilingual math dictionary
- AI-assisted math learning tools

## Technology Stack

- Next.js
- React
- Tailwind CSS
- TypeScript
- Firebase / Supabase (planned)

## Founder

Toby — Grade 10 student, math competition participant, and peer tutor.
